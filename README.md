# 🏎️ SCUDERIA INFINITY: ARCADE VELOCITY
**The Next Generation of High-End Arcade Racing for Unity.**

![Build Status](https://img.shields.io/badge/Unity-2022.3+-black?logo=unity)
![License](https://img.shields.io/badge/License-MIT-gold.svg)
![Version](https://img.shields.io/badge/Edition-Platinum_V1.0-red.svg)

Scuderia Infinity ist ein physikbasierter Arcade-Racer, der das intuitive Handling von Klassikern wie *Mario Kart* mit der Ästhetik moderner Luxus-Sportwagen verbindet. Entwickelt für maximale Performance und "Game Juice".

## ✨ Key Features
- **Advanced Drift System:** Dynamische Reibungsmanipulation für perfektes Kurven-Sliding.
- **Drift-Boost Engine:** Dreistufiges Aufladesystem für Geschwindigkeitsvorteile.
- **Arcade Physics:** Stabiles Handling durch tiefgelegten Schwerpunkt (CoM) und WheelCollider-Integration.
- **Luxury UI:** Minimalistisches, gold-akzentuiertes HUD für ein Premium-Gefühl.
