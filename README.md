# 🏎️ SCUDERIA PRESTIGE: INFINITY

![Version](https://img.shields.io/badge/Version-1.0.0-gold)
![Platform](https://img.shields.io/badge/Platform-iOS%20|%20Safari-black)
![License](https://img.shields.io/badge/License-Exclusive-red)

**SCUDERIA PRESTIGE** ist eine hochperformante Web-App-Erfahrung, die exklusiv für das iPhone entwickelt wurde. Es kombiniert minimalistisches Luxus-Design mit einer intensiven Arcade-Rennmechanik. 

Entwickelt für **Amiar Kempf**, definiert dieses Projekt die Grenzen dessen, was im mobilen Browser möglich ist, neu – durch den Einsatz von Hardware-beschleunigtem 3D-Rendering (Three.js) und synthetischem High-Fidelity Audio.

---

## 💎 Exklusive Features

* **Carbon & Gold Ästhetik:** Ein visuelles Erlebnis, inspiriert von modernen Hypercars.
* **Dynamic FOV System:** Die Kamera reagiert organisch auf die Geschwindigkeit – je schneller Sie fahren, desto intensiver wird der Tunnelblick.
* **Procedural Engine Sound:** Ein synthetischer Motor-Sound, der in Echtzeit auf die Beschleunigung reagiert (Web Audio API).
* **Glassmorphism UI:** Ein hochmodernes Head-Up-Display (HUD), das absolute Klarheit und Eleganz bietet.
* **Stage-Evolution:** Meistern Sie Checkpoints, um Zeitboni zu erhalten und in neue Geschwindigkeitsdimensionen vorzustoßen.

---

## 📱 Installation (iOS "App-Gefühl")

Um das volle **Prestige-Erlebnis** ohne störende Browser-Leisten zu genießen, folgen Sie diesen Schritten:

1.  Öffnen Sie den GitHub-Pages Link in **Safari** auf Ihrem iPhone.
2.  Tippen Sie auf das **Teilen-Symbol** (Viereck mit Pfeil nach oben).
3.  Wählen Sie **"Zum Home-Bildschirm hinzufügen"**.
4.  Starten Sie **SCUDERIA** direkt von Ihrem Homescreen wie eine native App.

---

## 🎮 Steuerung

Das Interface wurde für intuitive Daumen-Bedienung optimiert:

* **Mitte halten:** Beschleunigen (Engine Engage).
* **Linke Bildschirmhälfte:** Nach links steuern.
* **Rechte Bildschirmhälfte:** Nach rechts steuern.
* **Ziel:** Durchfahren Sie die goldenen Tore, um Ihr Zeitlimit zu verlängern.

---

## 🛠️ Technische Spezifikationen

* **Engine:** Three.js (WebGL)
* **Audio:** Web Audio API (Synthesizer)
* **Design:** CSS Glassmorphism & ACES Filmic Tone Mapping
* **Optimization:** Adaptive Resolution für stabile 60 FPS auf iOS Geräten.
